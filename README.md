# NemoClaw-agritwin-dairy-agentic-ai-platform-frontend

Frontend dashboard for AgriTwin Dairy Agentic AI Platform

## Views
- dashboard: Cow, Herd, HealthAlert
- analytics: MilkYield, Sensor
- recommendations: AgentRecommendation

## Pages
- FarmDashboard
- HerdTable
- CowDetails
- AlertsPanel
- AIAssistantPanel

## Shared Dependency
All pages import entities from `@robo-jedi-pl/agritwin-shared`.
